# Shifmon / シフモン

シフモンは、アルバイトのシフト管理・給料計算・カレンダー連携に、キャラクター育成要素を組み合わせたiPhone向けアプリです。

コンセプトは「働く予定を、ちょっと楽しみに変える」。

## アプリ概要

既存のシフト管理アプリを参考にしつつ、以下の2点で差別化する。

1. シフト・勤務時間・給料を管理するだけでなく、働いた成果がキャラクターや部屋の成長に反映される
2. ジョブカンやfreeeなどのシフト表スクリーンショットから、シフト情報を自動で読み取り、アプリとiPhone標準カレンダーに反映する

## 開発目的

アルバイトのシフト管理では、以下のような課題がある。

- もらったシフトを手入力するのが面倒
- 給料見込みは見えるが、働くモチベーションにつながりにくい
- 標準カレンダーへの転記が手間
- シフト管理アプリに楽しさや継続したくなる仕組みが少ない

シフモンでは、シフト管理を単なる記録作業ではなく、育成・達成感・可視化を通じて前向きな体験に変えることを目指す。

## 主な機能予定

### MVP機能

最初に実装する最低限の機能。

- 月間カレンダー表示
- シフト登録
- シフト編集
- シフト削除
- バイト先登録
- 時給設定
- 休憩時間設定
- 深夜時給設定
- 日給計算
- 月給見込み計算
- 月間勤務時間の表示
- SwiftDataによる端末内保存

### 次に実装したい機能

- iPhone標準カレンダー連携
- シフト登録時に標準カレンダーへ自動追加
- シフト変更時に標準カレンダーも更新
- シフト削除時に標準カレンダーからも削除
- スクリーンショットからのシフト自動読み取り
- 読み取り結果の確認画面
- キャラクター育成機能
- 働いた時間・稼いだ金額に応じた経験値付与
- アプリ内の部屋・装飾アイテム解放

## 開発方針

まずは自分用のiPhoneアプリとして完成させる。
その後、友人に使ってもらえるレベルまで改善し、最終的にはApp Store公開を目指す。

実装は小さく区切って進める。

1. 初期プロジェクト作成
2. ホーム画面作成
3. シフトデータモデル作成
4. シフト追加画面作成
5. シフト一覧表示
6. 月間カレンダー表示
7. 給料計算
8. 標準カレンダー連携
9. スクリーンショット読み取り
10. キャラクター育成機能
