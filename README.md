# Shifmon / シフモン

シフモンは、アルバイトのシフト管理・給料計算・カレンダー連携に、キャラクター育成要素を組み合わせたiPhone向けアプリです。

コンセプトは「働く予定を、ちょっと楽しみに変える」。

## アプリ概要

既存のシフト管理アプリを参考にしつつ、以下の2点で差別化する。

1. シフト・勤務時間・給料を管理するだけでなく、働いた成果がキャラクターや部屋の成長に反映される
2. ジョブカンやfreeeなどのシフト表スクリーンショットから、シフト情報を自動で読み取り、アプリとiPhone標準カレンダーに反映する

## 開発目的

アルバイトのシフト管理では、以下のような課題がある。

- もらったシフトを手入力するのが面倒
- 給料見込みは見えるが、働くモチベーションにつながりにくい
- 標準カレンダーへの転記が手間
- シフト管理アプリに楽しさや継続したくなる仕組みが少ない

シフモンでは、シフト管理を単なる記録作業ではなく、育成・達成感・可視化を通じて前向きな体験に変えることを目指す。

### 解決したエラー・詰まりポイント

- Xcode新規作成画面で Next が押せなかった
  - 原因：Organization Identifier が未入力、CloudKitが有効だった
  - 対応：`com.seimu9` を入力し、CloudKitは一旦OFFにする方針にした

- Gitコマンドで `fatal: not a git repository` が出た
  - 原因：ターミナルがShifmonプロジェクトのフォルダに移動できていなかった
  - 対応：`Shifmon.xcodeproj` があるフォルダへ移動してからGit操作を実行

- Apple IDログインでエラーが出た
  - 原因：Apple ID認証でパスワード入力に失敗
  - 対応：実機確認は後回しにし、まずはシミュレーターで開発を進める方針にした
