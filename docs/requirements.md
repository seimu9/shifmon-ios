# シフモン 要件定義書

## 1. プロジェクト概要

### 1.1 アプリ名

シフモン / Shifmon

### 1.2 コンセプト

シフモンは、アルバイトのシフト管理・給料計算・カレンダー連携に、キャラクター育成要素を組み合わせたiPhone向けアプリである。

コンセプトは「働く予定を、ちょっと楽しみに変える」。

### 1.3 開発目的

既存のシフト管理アプリでは、シフト登録・給料計算はできるが、以下の課題がある。

- もらったシフトを手入力するのが面倒
- 標準カレンダーへの転記が手間
- 給料見込みは見えるが、働くモチベーションにつながりにくい
- 継続して使いたくなる楽しさが少ない

シフモンでは、シフト管理を単なる記録作業ではなく、育成・可視化・達成感を通じて前向きな体験に変えることを目指す。

## 5. 非機能要件

### 5.1 操作性

- 直感的に操作できること
- シフト追加までの手数を少なくすること
- 入力ミスがあった場合は分かりやすく表示すること

### 5.2 保守性

- 機能ごとにファイルを分割する
- SwiftUIのViewが肥大化しすぎないようにする
- データモデルと画面表示を分ける

### 5.3 拡張性

- 将来的にiPhone標準カレンダー連携を追加できる設計にする
- スクリーンショット読み取り機能を後から追加できる設計にする
- キャラクター育成機能を後から拡張できる設計にする

## 9. テスト観点

### 9.1 シフト追加

- 勤務先を入力して保存できる
- 勤務先未入力でも保存できる
- 終了時間が開始時間より前の場合は保存できない
- 時給が未入力または0以下の場合は保存できない
- 休憩時間が未入力または不正な場合は保存できない
- 保存後にホーム画面へ戻る

### 9.2 給料計算

- 勤務時間から休憩時間を引いて計算される
- 時給と実働時間から見込み給料が表示される
- 今月分のみ月間合計に反映される

### 9.3 保存

- アプリを再起動しても登録済みシフトが残る
- 複数件のシフトを保存できる
